# Carbon Web Components Tabs - React Example

This example demonstrates how to use Carbon's dismissable tabs web component in a React application.

## Features

- Uses Carbon Web Components `<cds-tabs>` with dismissable functionality
- React state management for dynamic tab handling
- Reset functionality to restore default tabs

## Getting Started

```bash
npm install
npm run dev
```

## Usage

The example shows how to:
- Import and use Carbon web components in React
- Handle tab close events
- Manage tab state in React
- Dynamically render tabs and panels
